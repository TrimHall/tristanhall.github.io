Grit is a full-stack web application, created to motivate users to exercise. When registering on the platform, the user will join a "Tribe", based on how they work out. The options available are "Gargantuans" (body-builders), "All Naturals" (keen gym goers but not so weights focussed) and "Inbetweeners" (people who may not exercise so regularly). Users can create and adopt exercise programmes and track their daily progress. Completing the exercises or challenges set in the program will earn the user "Grit points" which contribute to their tribes overall performance. Progress is posted to a live feed and points accumulate on a data-visualisation dashboard.
